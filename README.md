# suyun-check
## 用途
用于签到
## 具体使用
```bash
  -checkUri string
        签到的路径资源 (default "/user/checkin")
  -level string
        日志等级 (default "info")
  -loginUri string
        登陆的路径资源 (default "/auth/login")
  -logposition string
        日志位置 (default "./app.log")
  -name-request string
        登陆用户名的参数,默认: email (default "email")
  -pass-request string
        登陆密码的参数,默认: passwd (default "passwd")
  -password string
        输入密码 (default "password")
  -protocol string
        http or https,默认: https (default "https")
  -urls string
        输入访问的urls,','隔开
  -username string
        输入用户名 (default "username")
```
