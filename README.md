# suyun-check
## 用途
用于签到
## 具体使用
```bash
  -checkUri string
        签到的路径资源 (default "/user/checkin")
  -level string
        日志等级 (default "info")
  -loginUri string
        登陆的路径资源 (default "/auth/login")
  -logposition string
        日志位置 (default "./app.log")
  -password string
        输入密码 (default "password")
  -username string
        输入用户名 (default "username")
```
